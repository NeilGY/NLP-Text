基于地图API的Web地图服务及应用研究  A Web Map Service and Application Based on Map API
              为解决企业和个人开展基于在线空间信息服务数据源以及技术问题,对基于地图API的Web地图服务及应用模式进行了研究.首先对企业和个人Web地图服务的需求和特点进行了分析,提出当前Web地图应用开发存在的问题及解决途径;其次,对开放式地图APi的特点、功能和工作原理进行了总结,提出了基于地图API的Web地图服务技术框架;最后,以一个校园地图服务应用系统为例开展了基于开放式API的Web地图服务应用实践.结果表明,地图API可解决网络地图服务数据源问题,节省开发成本,具有较好的应用前景.
